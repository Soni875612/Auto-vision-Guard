# 🎯 Executive Summary

Traffic congestion and road monitoring traditionally rely on manual observation, making them inefficient, expensive, and difficult to scale.

**AutoVision Guard Pro** is an AI-powered traffic monitoring and analytics platform that leverages **YOLOv8**, **OpenCV**, and **Flask** to detect vehicles and pedestrians from live camera feeds, images, and videos in real time.

The platform transforms raw visual data into actionable insights through intelligent traffic analysis, risk assessment, and interactive dashboards, making it suitable for smart city infrastructure, surveillance systems, and transportation analytics.

### Clone Repository

```bash

```

